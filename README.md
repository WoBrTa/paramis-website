# paramis-website

Official website for Paramis AI Strategy Consulting — [paramis.ai](https://www.paramis.ai)

## Pages
- **index.html** — Main landing page (hero, services, about, contact)
- **portal.html** — Client Portal login page (placeholder UI, no backend auth)

## Hosting
GitHub Pages from the `main` branch of this repository.

## Local Preview
```bash
python3 -m http.server 8080 --directory .
```
Then open `http://localhost:8080` in your browser.
